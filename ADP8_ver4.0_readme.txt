ADP8_ver4.0注意書き

・卒論で使用した最新版です
・ADP8_ver3.0との違いはデータベースへの保存方法の違いになります．
3.0では状態変数をすべて一列に並べているのに対し、4.0ではすべて状態変数ごとに行をわけています．
また4.0のほうが細かい部分で多少高速化が図られています。

<<使用方法>>
・control_cplexのmain.javaから実行してください
・コマンドプロンプトからも実行できます。

cmd、powershellなどでは「oplrun -D GET='0,1,1' main_loop.mod」で起動できます
GET='0,1,1'の部分ですが、'Iteration,手法，時点'の順番になってます
iterationは表示用なだけなので、どんな数字でも正常に動きます
手法は「0＝J1,J2両方」、「1=J1のみ」、「2=J2のみ」となります
時点は計算する時点番号を入れてください
